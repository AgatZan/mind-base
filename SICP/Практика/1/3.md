Определите процедуру, которая принимает в качестве аргументов три числа и возвращает сумму квадратов двух больших из них.
```scheme
(define  (^ x) (* x x))
(define (sum-of-num-that-max x y z)
	(if (> x y)
		(if (> y z)
			(+ (^ x) (^ y))
			(+ (^ x) (^ z))
		)
		(if (> x z)
			(+ (^ y) (^ x))
			(+ (^ y) (^ z))
		)
	)
)
```
