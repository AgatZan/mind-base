$$
f(n) = \begin{cases}

  n, & \text{if } n < 3, \\

  f(n-1)+f(n-2)+f(n-3), & \text{if } n \ge 3.

\end{cases}
$$
Рекурсивный
```scheme
(define (f n)
	(if (< n 3)
		n
		(+ (f (- n  1)) (f (- n 2) (f (- n 3))))
	)
)
```

Итеративный
```scheme
(define (f n)
	(define (f-iter f3 f2 f1 count)
		(if (= count 3)
			f3
			(f-iter (+ f1 f2 f3) f3 f2 (- count 1))
		)
	)
	(if (< n 4)
		n
		(f-iter 3 2 1 n)
	)
)

(f 4) = (+ f1 f2 f3)
(f 5) = (+ (+ f1 f2 f3) f3 f2)
```
