
### 1. **Единая точка входа**

`grammar.js`
```js
grammar({
	name: "test",
	extras:$=>[/\s/], // any space
	rules: $=>{
		document: $=>repeat(choice($.a,$.aa)), // if comment => dont work
	    a:$=>token('a'),
	    aa:$=>prec.right(2,'aa'),
	}
})
```
`result`
```scheme
(document [0, 0] - [1, 0]
  (a [0, 0] - [0, 1])
  (a [0, 2] - [0, 3]))
```
