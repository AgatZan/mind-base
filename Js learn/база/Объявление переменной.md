var - глобальная область видимости, а также моментальное объявление, как у функций, но без присвоения

let - локальная область видимости

const - неизменяемость сущности, но не внутренности. То есть 

```js
const a=3;
a=2; // Error!
const b={a:1,b:2};
b.a=3 // !(Error!)
```
